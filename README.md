# 功能



## **一 管理端**

### **1.1 电影管理**

- 支持excel 导入(豆瓣TOP250数据)和导出
- 支持手动添加
- 兼容豆瓣在线图片（处理防盗链） + 本地图片

### **1.2 排片管理**

- 选择电影厅、场次（时间段）、价格然后该电影就上线了，否则用户是只能查看电影信息，没办法购买的

## **二 用户端 （网站/H5移动端）**

- 未上映电影无法买票
- 用户选择电影、影厅、场次后可以购票，如果卖掉的票，座位是红色的，无法点击
- 买票使用支付宝测试环境沙箱支付
- 买票后可以评价，评价的数据会影响协同过滤推荐
- 推荐是mahout协同过滤
